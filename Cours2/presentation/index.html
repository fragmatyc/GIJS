Qu'est-ce?

Angular.js est une librairie de scripts.

Exemple

<div class="panel-heading">
  <h1>Angular MVC Tutorial <small>Tic Tac Toe </small></h1>
</div>
<div class="panel-body">
  <div class="col-sm-6">
    <tic-tac-toe-grid></tic-tac-toe-grid>
  </div>
  <!-- Etc... -->

Les directives permettent de créer de nouvelles balises HTML!

<tic-tac-toe-gird></tic-tac-toe-grid>

À quoi ça sert?

Yuck!

<ul>
  <script>
    for (i = 0; i < 10; i++) {
      document.write('<li>Item #' + i + '</li>');
    }
  </script>
</ul>

Pourquoi Angular?

  • HTML est un language déclaratif.
  • SoC, SRP (SOLID), Cohésion, autre?
  • Le code des vues PEUT être propre!

Angular offre:

  • La création de modules
  • Directives
  • Controllers
  • Services/Factories

Module

Un module représente une feature de l'application, un composant réutilisable.

Un système doit avoir un module d'application (orchestration des dépendences).

Création d'un nouveau module?!

var monModule = angular.module('monModule', []);

Utilisation d'un module

<div ng-app="monModule"></div>

Que c'est?! ng-app?

<div ng-app="monModule"></div>

Une directive!

ngApp doc!

Controller

Un controller prend en charge la gestion des événements de la vue et délégue l'exécution des règles d'affaires à la couche modèle.

Création d'un controller?!

var monModule = angular.module('monModule', []);
                        
monModule.controller('monController', function ($scope) {
  // Code du controller
});

Utilisation du controller

<div ng-app="monModule" ng-controller="monController">
                        
</div>

Service

Un service est un objet substituable/réutilisable qui permet l'exécution de la logique d'affaire d'un module.

Création d'un service?!

var monModule = angular.module('monModule', []);
                        
monModule.service('hiService', function () {

});

Utilisation du service

monModule.controller('monController', function ($scope, hiService) {
                        
});

Directive

Les directives sont des marqueurs pour le DOM qui seront compilés en HTML par le service $compile d'Angular.

Création d'une directive

var monModule = angular.module('monModule', []);
                        
monModule.directive('customTag', function () {
  return {
    restrict: 'E',  // Element seulement
    template: "
Hello {{who}}
"
  };
});

Utilisation de la directive

<div id="applicationContainer" ng-app="monModule"
    ng-controller="monController">
  <custom-tag></custom-tag>
</div>
