Le système de grille

Important à savoir:

  • container vs container-fluid
  • row
  • col-xx-#

Exemple

<div class="container-fluid">
    <div class="row">
        <div class="col-sm-6 demo-cell">Contenu1</div>
        <div class="col-sm-6 demo-cell">Contenu2</div>
    </div>

    <div class="row">
        <div class="col-sm-4 demo-cell">Contenu1</div>
        <div class="col-sm-4 demo-cell">Contenu2</div>
        <div class="col-sm-4 demo-cell">Contenu3</div>
    </div>
</div>

Angular Routes!

Permet de faire la navigation dans une SPA.

bower install angular --save
bower install angular-route --save

Injection de angular-route

var monModule = angular.module('monModule', ['ngRoute']);

Définition d'un route provider

var monModule = angular.module('monModule', ['ngRoute']);

monModule.config(function ($routeProvider) {
    $routeProvider
        .when('/uriPath', {
            templateUrl: 'pathToView.html',
            controller: 'nomOfControllerToUse'
        })
        .when('/otherUriPath', {
            templateUrl: 'pathToOtherView.html',
            controller: 'otherController'
        })
        .otherwise({
            redirectTo: '/uriPath'
        });
});
